## Ordnerstruktur

Das Projekt verwendet die folgende Ordnerstruktur. Der Ordner "Data" und dessen Struktur sind zunächst nicht im Repository hinterlegt, beschreiben jedoch den entstandenen Aufbau, sobald die Skripte auf die genutzten Daten ausgeführt werden.

``` txt
Repository Root
│── README.md                    # Projektübersicht und Reproduzierbarkeit
│── Thesis.pdf                   # Finaler Bericht
│── Disputation.qmd              # Endpräsentation
│── Disputation.html             # Präsentationsausgabe
│── customstyle.css              # Stilvorlagen
│── Settings.R                   # Globale Einstellungen
│── Plot-Styling.R               # Standardisiertes Plot-Design
│
├── Data/                        # Datensätze
│   ├── Raw/                     # Rohdaten
│   └── Intermediate/            # Weiter bearbeitete Datensätze
│
├── Programs/                    # Skripte zur Datenaufbereitung, Analyse und Modellierung
│   ├── ...
│   └── ...
│
├── Plots/                       # Finale Grafiken für Bericht/Präsentation aus den Daten
└── Images/                      # Zusätzliche manuell erstellte Visuals
```
